# 🚀 Com utilitzar aquest repositori

1. Entra a la carpeta del teu **cicle (DAM o DAW)**.
2. Selecciona el **curs** corresponent.
3. Obre la carpeta de la **assignatura**.
4. Descarrega o clona els recursos que necessitis.

# 📁 Com descarregar només una carpeta

GitHub no permet descarregar carpetes individuals directament.

Opció recomanada:

1. Copia l'URL de la carpeta del repositori
2. Ves a https://downgit.github.io/
3. Enganxa l'URL
4. Prem **Download**
